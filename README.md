## HibikiUEProject

This is a working Unreal Engine project created from the UE4SS dump.

## Build requirements
1. Unreal Engine 4.27
2. Visual Studio 2019 with following workloads:
   - Desktop development with C++
   - MSVC v142 toolset
   - Windows 10 SDK

To build the project:
1. Open project folder
2. Right-click `Hibiki.uproject` file and select "Generate Visual Studio project files"
3. Open generated solution (`Hibiki.sln`)
4. Build in `Development Editor | Win64` configuration
